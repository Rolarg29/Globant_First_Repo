Git init -- inicializa un repositorio local en el directorio donde esté

Git status -- sirve para revisar que archivos ya estan listos o no en el area de preparación antes de hacer commit

Git add . -- agrega todos los archivos pendientes al area de preparación

Git add <nombre_archivo> -- agrega un archivo o directorio escpecifico al staging area

Git commit .m "<mensaje>" -- sirve para guardar y hacer commit con un mensaje a los cambios actuales

Git push -- sube los cambios del repo local al remoto en Github(u otra)
